## Commands (using npm run)
- **build** - Uses ng build to build the project into the 'dist' folder.
- **start** - Uses ng serve to run the application.
- **lint** - Lints the application.
- **prettier** - Styles the application.

## Other commands
- **ng generate component <component-name>** - Generates an Angular component.
- **ng generate --help** - Help.
